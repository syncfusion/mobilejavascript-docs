# Disable rating

The "data-ej-readonly" attribute is used to disable the interaction in the Rating control.The default value is false.

{% highlight html %}

<div id="rating_sample" data-role="ejmrating" data-ej-readonly="true">

</div> 

{% endhighlight %}
