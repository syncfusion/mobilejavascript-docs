# Customize orientation

You can customize the orientation the rating using **data-ej-orientation** attribute . It can be,

1. Horizontal.
2. Vertical. 

{% highlight html %}

<div id="rating_sample" data-role="ejmrating" data-ej-orientation="vertical">

</div>

{% endhighlight %}
